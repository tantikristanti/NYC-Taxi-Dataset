## 📚 Dataset Repository

This repository provides a collection of ready-to-use datasets for:

- **Structured Data:** Ideal for exploring and testing relational databases such as PostgreSQL and MySQL, or for building applications with SQL backends.
- **Unstructured Data:** Ready to leverage in Large Language Model (LLM) projects, including search engines, Retrieval-Augmented Generation (RAG) pipelines, and chatbot development.
- **Proof of Concept (PoC):** Accelerate a validation process with sample data to quickly prototype, demonstrate, and test new ideas.

Complete source attributions and references can be found in the [References](#references) section.

### 📂 About the Datasets

All datasets in this repository are collected and curated from publicly available online sources, including official websites, open GitHub repositories, and Kaggle. Each dataset is organized in its own dedicated folder, containing the data files along with a description, schema, and
usage notes.

### 🔧 Available Formats

To support flexible use across different platforms and workflows, the datasets are provided in both **JSON** and **CSV** formats. This makes them easy to import into databases, integrate into applications, or load into data analysis tools.
